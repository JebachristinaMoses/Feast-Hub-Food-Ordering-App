# 🍽️ Feast Hub - Food Ordering App

A modern **Full-Stack Food Ordering Web Application** that allows customers to browse menus, place food orders, and manage their accounts through an intuitive interface. The application also provides an admin dashboard for managing food items and orders.

## 📖 Project Overview

Feast Hub is a responsive food ordering platform developed using modern web technologies. The application provides a seamless ordering experience for customers while enabling administrators to efficiently manage food items, customer orders, and user information.

## ✨ Features

### 👤 Customer

- User Registration & Login
- Browse Food Menu
- Search Food Items
- Add Items to Cart
- Place Orders
- View Order History
- Responsive User Interface

### 🔐 Admin

- Admin Login
- Dashboard
- Add New Dishes
- Update Food Items
- Delete Food Items
- Manage Customer Orders

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/JebachristinaMoses/Feast-Hub-Food-Ordering-App.git
```

### Navigate to the project

```bash
cd Feast-Hub-Food-Ordering-App
```

### Install Frontend Dependencies

```bash
npm install
```

### Install Backend Dependencies

```bash
cd server
npm install
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd server
npm start
```

### Start Frontend

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
